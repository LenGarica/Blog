# 文献阅读记录

### 1.第一篇

| 文章名       | A TRANSPORT MODEL FOR WATER-SOLUBLE CONSTITUENTS OF SYNTHETIC OIL SPILLS IN RIVERS |
| :----------- | ------------------------------------------------------------ |
| **作者**     | STEPHEN E. HERBES* and GOUR-TSYH YEH Environmental Sciences Division, Oak Ridge National Laboratory Oak Ridge, Tennessee 37831 |
| **摘要**     | 开发了一个数值模型，用于预测合成油意外泄漏到河流中导致的酚类污染物的溶解水浓度。该模型（称为SOPTRAN，用于合成石油污染物输送）使用隐式积分隔室法同时求解油膜扩散、石油蒸发、酚类溶解、平流和扩散过程的算法。为了测试SOPTRAN，将预测值与将煤液化产品应用于两个3.75 x 3.75 x 1 m池塘后酚类溶解到下层水中的测量值进行比较。酚类物质溶解过程中测得浓度的平均值为预测值的84%（变异系数= 43%）。通过最小限度的校准，SOPTRAN充分描述了苯酚和C1-至C1-烷基苯酚异构体基团的不同溶解速率。该模型用于模拟假设的300-m3（80，000-gal）煤液从驳船意外泄漏到大型通航河流中，包括瞬时和非瞬时泄漏（即：泄漏）溢出情形。 |
| **关键词**   | Transport model 、Synthetic oil、Phenolics、Dissolution      |
| **研究目的** | 使用隐式积分隔室法同时求解油膜扩散、石油蒸发、酚类溶解、平流和扩散过程的算法。 |
| **研究方法** |                                                              |
| **参考部分** | 河流中大量溢油的运动主要受纵向平流影响。海岸线污染实际上不可避免。伊尔内费尔。海洋中的湍流比可航行的河流中的湍流更大;因此，油滴湍流分散到下面的水中（即，乳化）在河流溢漏中可能远不如在海洋溢漏中重要。 |
